JAVASOLT 6B TODO
================

TODO szöveg:

  [6B] MANUAL_CHECKPOINT event propagation

  A TestRunner._run_manual_checkpoint() hívjon on_event callbacket
  MANUAL_BQ_CHECKPOINT_REACHED eventtel, hogy a TestRunnerWorker checkpoint_reached

      "manual_bq_checkpoint_reached": True,
  }


JAVASOLT IMPLEMENTÁCIÓ 6B-BEN
=============================

A _run_manual_checkpoint(step) belsejében:

  event = {
      "event_code": "MANUAL_BQ_CHECKPOINT_REACHED",
      "event_message": (
          "BQ learning fizikai ciklus kézi ellenőrzési pontja elérve."
      ),
      "step_name": step.label,
      "status": "WAITING_FOR_MANUAL_CHECKPOINT",
      "manual_bq_checkpoint_reached": True,
  }

  self.logger.log_event(event)

  if self.on_event:
      self.on_event(event)

  return TestResult(
      status="STOPPED",
      reason="MANUAL_BQ_CHECKPOINT_REACHED",
  )

Vagy ha a runner később várakozó állapotban marad:

  return TestResult(
      status="WAITING_FOR_MANUAL_CHECKPOINT",
      reason="MANUAL_BQ_CHECKPOINT_REACHED",
  )
